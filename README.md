# Install

## gradle

```gradle
    compile 'topol777:android-library:1.35.5'
```

## maven

```xml

<dependency>
    <groupId>topol777com.github.topol777</groupId>
    <artifactId>android-library</artifactId>
    <version>1.35.5</version>
    <type>aar</type>
</dependency>
```
